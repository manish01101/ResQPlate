# ResQPlate Frontend

This is the user interface for **ResQPlate**, built with React.js. It provides separate dashboards for Donors and NGOs and visualizes food availability on an interactive map.

## Tech Stack
- **Framework:** React.js (Create React App)
- **Styling:** Tailwind CSS (or CSS Modules)
- **Maps:** React-Leaflet / Leaflet API
- **HTTP Client:** Axios
- **Routing:** React Router DOM
- **Notifications:** React-Toastify
