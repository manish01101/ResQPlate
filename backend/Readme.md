# ResQPlate Backend (API Server)

This is the server-side application for **ResQPlate**, built with Node.js, Express, and MongoDB. It handles authentication, database management, and geo-spatial logic for food donations.

## Tech Stack
- **Runtime:** Node.js
- **Framework:** Express.js
- **Database:** MongoDB Atlas (Mongoose ODM)
- **Authentication:** JWT (JSON Web Tokens) & BcryptJS
- **Image Handling:** Multer & Cloudinary (Optional)
